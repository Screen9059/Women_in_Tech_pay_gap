# Women_in_Tech_pay_gap
Beginner data analysis project exploring gender pay differences using Kaggle dataset.


## 📊 Project Overview
This is a beginner data analysis project that explores gender pay differences in the tech industry using a Kaggle dataset.

---

## 🎯 Objectives
- Understand gender distribution in tech jobs
- Analyze salary differences between genders
- Explore how education and job roles affect salary
- Practice basic data cleaning and visualization

---

## 📁 Dataset
The dataset used in this project is from Kaggle:
Glassdoor Gender Pay Gap dataset

---

## 🧰 Tools Used

- Excel
- Python
- Pandas
- SQL
- Tableau

---

## 🔍 Key Questions
- What is the gender distribution in the dataset?
- Is there a salary difference between men and women?
- Which departments have higher salaries?
- Does education level affect salary?

---

## 📈 Analysis Steps
1. Load dataset
2. Clean missing values
3. Explore gender distribution
4. Analyze salary differences
5. Create visualizations
